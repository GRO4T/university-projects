## Stała liczba iteracji

Dla każdej z tych opcji zebrać wynik i czas działania

Wymiarowości do sprawdzenia:

- n = 20
- n = 50
- n = 100
- n = 200

liczba iteracji:

dla losowego:
1000000

dla roju:
10000 / n

100*n
